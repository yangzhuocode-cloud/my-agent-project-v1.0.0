## 快速搜索指南

### 按关键词搜索

使用以下命令在所有文档中搜索关键词：

```bash
# 搜索 token 相关内容
grep -r "token" docs/issues/

# 搜索大模型相关内容
grep -r "大模型" docs/issues/

# 搜索 Git 相关问题
grep -r "Git" docs/issues/problems/
```

### 按文件类型搜索

```bash
# 查看所有问题文档
ls docs/issues/problems/**/*.md

# 查看所有笔记文档
ls docs/issues/notes/**/*.md

# 查看所有想法文档
ls docs/issues/ideas/**/*.md
```

## 维护说明

### 添加新文档时

1. 在文档开头添加 YAML metadata：
```yaml
---
keywords: [关键词1, 关键词2, ...]
created: YYYY-MM-DD
type: 问题记录/技术笔记/想法
summary: 一句话概括文档内容
---
```

2. 在本索引文件中添加条目：
   - 在"按主题分类"中找到合适的分类添加
   - 在"按时间排序"中添加时间条目

### 索引更新频率

- 每次创建新文档时立即更新索引
- 每周检查一次索引完整性
- 每月整理一次分类结构
