# My First Agent 技术演进规划

**日期**：2026-03-19  
**类型**：技术规划  
**优先级**：中

---

## 当前状态

My First Agent 目前只有一个最简单的 Python 脚本（`agents/my first agent/main.py`），实现了基本的 API 调用和对话功能。

## 技术演进计划

### 第一阶段：Python 版本完善

**目标**：使用 Python 完成最简单的框架

**需要添加的功能**：
1. 上下文管理
2. 上下文切割
3. 其他基本逻辑

**语言**：Python

**原因**：先用 Python 快速实现和验证功能

### 第二阶段：Go 语言重写

**目标**：将整个 Agent 用 Go 语言重写

**时机**：完成 Python 版本的最简单框架之后

**语言**：Go

**原因**：以后的开发全部打算用 Go

## 技术路线

```
当前状态（Python 简单脚本）
    ↓
第一阶段：Python 版本完善
    - 添加上下文管理
    - 添加上下文切割
    - 完善基本框架
    ↓
第二阶段：Go 语言重写
    - 将所有功能用 Go 重写
    - 作为第二版本
    ↓
后续开发
    - 全部使用 Go 语言
```

## 备注

这是一个技术演进规划，先用 Python 快速验证功能，再用 Go 重写以获得更好的性能和维护性。
